## Highlights

- Developed collaboration recommendation algorithm using Pinecone and cosine similarity
- Implemented automatic collaboration proposal generation using OpenAI API
- Designed database for alumni company info, matching results, and collaboration records
- Developed APIs for company info input, recommendation results, and proposal generation

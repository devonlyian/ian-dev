System that automates document approval process by integrating RPA with server for approval requests, processing, and document number issuance

## Highlights

- Document info and approver list storage and management using Redis
- Implemented automatic email notification to next approver during approval process
- Created approval request email templates using JavaMail API and Thymeleaf
- Integrated with Brity RPA for automatic document number issuance upon final approval
